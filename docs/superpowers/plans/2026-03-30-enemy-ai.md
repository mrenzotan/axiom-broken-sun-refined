**Goal:** Add a plain C# `EnemyActionHandler` class that attacks the player on `EnemyTurn`, and wire it into `BattleController` so the battle state machine reaches `Victory` or `Defeat`.

**Architecture:** `EnemyActionHandler` is a plain C# class (no MonoBehaviour) that mirrors `PlayerActionHandler` — it takes two `CharacterStats` at construction and exposes `ExecuteAttack()`. `BattleController` (the MonoBehaviour wrapper) instantiates it alongside `PlayerActionHandler` and calls it when `BattleManager` fires the `EnemyTurn` state. No new asmdef is needed — both new files live in folders that already have asmdefs.

```csharp
using System;

namespace Axiom.Battle
{
    /// <summary>
    /// Plain C# class that executes the enemy's attack action during EnemyTurn.
    /// Reads stats from CharacterStats instances injected at construction time.
    /// No MonoBehaviour — BattleController creates and holds this.
    /// </summary>
    public class EnemyActionHandler
    {
        private readonly CharacterStats _enemyStats;
        private readonly CharacterStats _playerStats;

        public EnemyActionHandler(CharacterStats enemyStats, CharacterStats playerStats)
        {
            _enemyStats  = enemyStats;
            _playerStats = playerStats;
        }

        /// <summary>
        /// Deals damage to the player using the formula: damage = max(1, enemyATK - playerDEF).
        /// Returns true if the player is defeated (CurrentHP == 0).
        /// </summary>
        public bool ExecuteAttack()
        {
            int damage = Math.Max(1, _enemyStats.ATK - _playerStats.DEF);
            _playerStats.TakeDamage(damage);
            return _playerStats.IsDefeated;
        }
    }
}
```

```csharp
private void HandleStateChanged(BattleState state)
{
    Debug.Log($"[Battle] → {state}");

    if (state == BattleState.EnemyTurn)
        ExecuteEnemyTurn();

    if (state == BattleState.Fled)
    {
        // Phase 4: GameManager will handle the scene transition.

private void ExecuteEnemyTurn()

}
```
